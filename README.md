# Axis Dashboard UI

## Features
### Teachers
1. Record attendance of students
2. View attendace history of students

### Admin
1. View attendance history of all students
2. View computed payments for teachers
3. Term Details
    - Modify term end date
    - Pre-allocate sessions for next term in advance
    - Students who have unallocated sessions are highlighted in yellow
4. Add/withdraw students to/from classes
